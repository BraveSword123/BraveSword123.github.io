My Favorite Music Player

My Favorite Music is a personal web player that showcases five of my favorite songs in a visually themed experience. I built it with HTML, CSS, and vanilla JavaScript, designing a custom audio UI (play/pause, next/prev, seek bar) that syncs with the native Audio API. Under the hood, tracks are stored in a doubly circular linked list for intuitive next/previous navigation. A single changeSong() function updates album art, audio sources, backgrounds, and control colors per track, while event listeners keep the progress slider in sync and reset the UI when playback ends. The interface uses Font Awesome icons, a “Digital Numbers” display for the Now Playing panel, and @media queries for responsive layouts.

Features
  • Custom audio player with play/pause, forward, and backward controls
  • Song-specific visuals and color palettes
  • Scrollable seek bar for audio playback progress
  • Doubly circular linked list for intuitive song navigation
  • Responsive layout and embedded fonts/icons via CDN
  • Includes a repeat button that allows users to play the song on loop
  • Responsive Design – @media Query
